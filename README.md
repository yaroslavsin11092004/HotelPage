## MVP-проект системы бронирования номеров (ASP.NET Core + React)

### О проекте
Веб-приложение для автоматизации бронирования номеров в отеле. Реализован базовый REST API для добавления бронирований, поиска свободных номеров на выбранные даты и отображения текущей занятости фонда.

### Архитектура и технологии
Проект разделен на независимый бэкенд (серверную часть) и легковесный фронтенд-клиент (SPA).

*   **Backend:** C#, .NET Core (ASP.NET Core Web API).
*   **Database & ORM:** PostgreSQL, Entity Framework Core.
*   **Архитектурный паттерн:** Выделен слой контроллеров (API), сущностей базы данных (Models) и реализован паттерн Repository (Репозиторий) для абстракции работы с данными.
*   **Frontend:** React, чистый TypeScript (без использования сторонних UI-библиотек). Разработаны базовые интерактивные формы для взаимодействия с API.
